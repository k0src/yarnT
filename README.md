**yarnT** is a tiny terminal based text editor.

**Features:**

- Syntax Highlighting
- File I/O
- Search Functionality

**Usage:**

- `yarnT <filename>` - Open/Create a file.
- `Ctrl+W` - Jump to next word.
- `Ctrl+F` - Find in file.
- `Ctrl+S` - Save.
- `Ctrl+Q` - Quit.

yarnT is created following [this](https://viewsourcecode.org/snaptoken/kilo/) tutorial.
